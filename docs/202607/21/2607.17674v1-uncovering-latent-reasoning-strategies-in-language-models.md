## 摘要
一个在推理任务上训练的语言模型 $p_θ(y \mid x)$ 通过多种不同策略学习解决问题，但这些策略隐含在模型的响应分布中且相互纠缠。我们研究将给定预训练语言模型的响应分布分解为结构化的、策略条件化表示的问题。具体而言，我们学习一个潜在变量分解 $p_θ(y \mid x) \leadsto (r_φ(z \mid x), g_φ(y \mid x,z))$，其中路由器 $r$ 将每个输入映射到潜在策略 $z$ 上的分布，生成器 $g$ 则基于该策略产生响应。一个关键挑战在于，从基础模型初始化的生成器已经表示了不使用 $z$ 的 $p_θ(y \mid x)$。因此，标准变分推断不鼓励模型通过 $z$ 路由信息，并可能导致严重的后验坍塌形式。为解决这一问题，我们提出一个变分目标，该目标测量相对于基础模型响应损失的分数信息增益，并将重构压力集中在具有高基础模型惊奇度的标记上，从而鼓励 $z$ 编码与策略相关的响应变异。我们引入一个多策略算法任务的基准，并表明该目标在保持基础模型响应分布的同时，恢复出与不同参考策略对齐的潜在编码。

## Abstract
A language model $p_θ(y \mid x)$ trained on reasoning tasks learns to solve problems via multiple distinct strategies, yet these strategies are implicit and entangled within the model's response distribution. We study the problem of decomposing the response distribution of a given pretrained language model into a structured, strategy-conditioned representation. Specifically, we learn a latent-variable factorization $p_θ(y \mid x) \leadsto (r_φ(z \mid x), g_φ(y \mid x,z))$, where a router $r$ maps each input to a distribution over latent strategies $z$ and a generator $g$ produces the response conditioned on that strategy. A key challenge is that the generator, initialized from the base model, already represents $p_θ(y \mid x)$ without using $z$. Standard variational inference therefore gives the model no incentive to route information through $z$ and can yield a severe form of posterior collapse. To address this, we propose a variational objective that measures fractional information gain relative to the base model's response loss and concentrates reconstruction pressure on tokens with high base model surprisal, encouraging $z$ to encode strategy-relevant response variation. We introduce a benchmark of multi-strategy algorithmic tasks and show that this objective recovers latent codes aligned with distinct reference strategies while preserving the base model's response distribution.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：大型语言模型在推理任务中会隐式学习多种不同的解决策略（如归纳证明、矛盾证明、不同排序算法等），但这些策略在模型的响应分布中相互纠缠，没有显式的变量来标识或控制生成时使用的策略。现有的潜在变量生成模型（如VAE）无法直接从预训练模型中提取这种策略结构，因为预训练模型本身已经实现了响应分布，导致标准变分法会产生“后验坍塌”（latent collapse）——模型倾向于忽略潜在变量而直接复制基模型分布。

- **核心问题**：如何在不改变基模型响应分布的前提下，将其分解为路由器（router）和策略条件生成器（generator）的因子化形式，使得潜在变量 `z` 能捕获策略级别的变异，并且在不同输入间具有语义一致性。

- **整体含义**：恢复策略结构能够将隐式策略变为显式对象，从而支持模型可解释性、可控生成、以及强化学习中的定向探索（如探索低采样策略）。这项工作为理解和干预语言模型的推理行为提供了新框架。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：学习一个路由器-生成器因子化模型 `(r_φ(z|x), g_φ(y|x,z))`，使 `p_φ(y|x) = ∫ r_φ(z|x) g_φ(y|x,z) dz ≈ p_θ(y|x)`，同时让 `z` 具有信息性（编码超越 x 预测性的变化）和策略对齐性（同一潜在值在不同输入上代表相同策略）。

- **关键挑战**：基模型 `p_θ` 已经能生成响应，而标准ELBO只要求边际分布匹配，会鼓励模型忽略 `z`。作者提出**模型导向的重构压力**（Model-Directed Reconstruction Pressure），利用基模型自身的惊讶度（surprisal）来指导训练。

- **关键技术细节**：
  - 架构：基于预训练Transformer（如Qwen2.5）添加路由头（输出高斯分布均值和方差）和潜在嵌入投影。路由器读取输入生成 `z`，生成器在生成时通过伪token嵌入 `E(z)` 进行条件化。
  - 训练时使用独立的后验网络 `q_ξ(z|x,y)` 推断潜在变量，仅在训练阶段使用。
  - **损失函数**：重构项 `R_θ` 被基模型的平均惊讶度归一化，并使用逐token权重 `a_θ,t`（由基模型惊讶度指数加权，参数 γ 控制集中程度）。总体目标：`J_θ = R_θ + β·KL(q_ξ∥r_φ)`，其中 `R_θ` 定义为 `(1/c_θ) E[∑ a_θ,t · (-log g_φ(y_t|x,z,y<t))]`，`c_θ` 是基模型平均损失。这使损失具有“分数信息增益”解释。
  - 逐token加权策略高惊讶度位置（即基模型不确定性大的分支点），迫使 `z` 编码策略相关信息。

- **算法流程**：初始化ϕ和ξ为θ，冻结pθ；估计归一化常数cθ；在每次训练步中从基模型采样响应y，计算逐token权重，从后验采样z，计算损失并梯度更新ϕ和ξ。
